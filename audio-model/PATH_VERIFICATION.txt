================================================================================
PATH CONSTRUCTION METHOD
================================================================================

All paths are built using Python's Path object with the / operator:

  ✓ Path(__file__).parent               → Gets current file's directory
  ✓ PROJECT_ROOT / 'data'               → Builds path relative to PROJECT_ROOT
  ✓ DATA_DIR / 'sessions'               → Builds path relative to DATA_DIR
  ✓ session_path / 'component1'         → Builds path relative to session

This works on ALL operating systems (Windows, Mac, Linux) automatically!
