SETUP FOR SOMEONE FETCHING THIS BRANCH
================================================================================

Step 1: Clone Repository
  $ git clone <repo-url>
  $ cd CareBnB
  $ git checkout audio-model

Step 2: Navigate to Audio-Model
  $ cd audio-model

Step 3: Install Dependencies
  $ pip install -r requirements.txt

Step 4: Configure API Keys
  $ cp .env.example .env
  $ nano .env  # Add your OpenAI and Anthropic API keys

Step 5: Fine-Tune Model (ONE TIME, 10-30 min)
  $ python -c "from src.models.component2 import fine_tune_model; fine_tune_model()"

Step 6: Use the Pipeline
  $ python
  >>> from pipeline_api import PipelineAPI
  >>> api = PipelineAPI()
  >>> result = api.process_audio(audio_bytes, format='m4a')
  >>> print(result.status)
